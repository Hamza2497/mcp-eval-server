Five evaluation tools

📋
score_against_rubric
Score an output against a custom rubric with weighted criteria and a passing threshold.
✅
evaluate_factual_accuracy
Check whether an output contradicts or misrepresents a set of known facts.
🎯
check_relevance
Check whether an output actually addresses what the user asked.
🔍
check_logical_consistency
Detect internal contradictions within an output.
⚖️
compare_outputs
Compare two LLM responses to the same query and determine which is better and why.
